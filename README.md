# 𝚖𝚘𝚗𝚘𝚝𝚎𝚡𝚝
A small web app that turns a standard text into unicode 𝚖𝚘𝚗𝚘𝚝𝚎𝚡𝚝.

> **https://codingunicorn.dev/monotext**

Use it to format code on your Instagram and Twitter. Like this:

```
  𝚛𝚎𝚝𝚞𝚛𝚗 𝙰𝚛𝚛𝚊𝚢
    .𝚏𝚛𝚘𝚖(𝚝𝚎𝚡𝚝)
    .𝚖𝚊𝚙(𝚌 => 𝚜𝚙𝚎𝚌𝚒𝚊𝚕(𝚌))
    .𝚖𝚊𝚙(𝚌 => 𝚌𝚑𝚊𝚛(𝚌))
    .𝚖𝚊𝚙(𝚌 => 𝚗𝚞𝚖(𝚌))
    .𝚖𝚊𝚙(𝚌 => 𝚌)
    .𝚓𝚘𝚒𝚗('');
```
